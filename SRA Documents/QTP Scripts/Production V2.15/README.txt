README.TXT for QTP
================================================================================================================

Before Execution for each Release, Kindly make sure that you have done the below things:

1. Set the Environment in TestSuite file - Testdata sheet.
2. Do the DB Refresh(www-qaindia.razoo.com/admin/db_refresh).
3. In the "TestSuite" (Tesdata - sheet), please set the disbursement date as today
4. Add the following brands in the host file of QTP execution machine - subdomain11-qaindia.razoo.com and subdomain1-qaindia.razoo.com
5. Fill 'Yes' value for the Scenarios to be executed in TestSuite file - Tests sheet.
6. Open the Driver Script in QTP window.
7. Click on 'Run' button.
================================================================================================================
